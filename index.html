FM Radio RDS webapp

Previous Next

Este proyecto forma parte de mi trabajo de fin de grado universitario. Consiste en una aplicación web que, en función de la geoposición del usuario, devolverá el listado de emisiones cercanas a su posición.

Desarrollada principalmente para el sistema operativo móvil Firefox OS, se ha adaptado y reconvertido en una aplicación web accesible desde cualquier dispositivo y plataforma. De este modo, se logra ser de utilidad para un número de usuarios prácticamente infinito.

La aplicación web FM Radio RDS está en continua mejora. Se espera la colaboración objetiva y positiva de los usuarios, en cuanto a las puntuaciones realizadas como los nuevos datos registrados en el sistema, para el crecimiento de la webapp.

  • Publicado en 2015

Sobre mí
